Excel / CSV → Code128 barcodes (ZIP)

Upload .xlsx, .xls, or .csv with columns: S No, SKU Code, Code. Code can look like SN: 2603-CVMJ3-512850.

Notes
  • Barcodes are Code128, saved as PNG (472×177 px, same as your Python script).
  • Each file is named from the barcode value (e.g. 2603-CVMJ3-512850.png).
  • CSV must use comma-separated fields; quoted values (e.g. with commas inside) are supported. UTF-8 BOM is OK.
  • Everything runs in your browser; the file is not uploaded to any server.
